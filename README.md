## 📱 App Screens

The mobile application features a clean, intuitive UI structured around the following core screens:

* 📸 **Camera & AI Scan Screen:** Integrates with Azure Document Intelligence to capture and scan student IDs seamlessly.
* 📦 **Inventory Screen:** Provides a real-time overview of all available sports equipment.
* 🛒 **Checkout Screen:** Streamlines the process of assigning equipment to verified students.
* 🔄 **Return Screen:** Handles equipment returns and automatically updates inventory levels in the cloud.
* 📜 **Transaction History Screen:** Displays a comprehensive log of all past checkout and return activities.
* ⚙️ **Sports Management Dashboard:** The central hub for administrators to navigate and manage the system.
